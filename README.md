# website-for-mimi

Interactive static website for **Little Beans Café Club**.

## Pages
- `index.html` – main landing page with interactive planner, rotating testimonials, and booking form feedback.
- `activities.html` – child activity overview.
- `community.html` – community events and quick links.

## Run locally
```bash
python -m http.server 8000
```
Then open `http://localhost:8000`.
